# 🎈 Landing Page - Balloon Tour

Учебный лендинг для туров на воздушном шаре. Проект на стадии разработки - реализована только hero-секция с анимациями, остальные секции ещё не вёрстаны.

---

## 🚀 Функционал

- Hero-секция с фоновым изображением и градиентным оверлеем
- `clip-path` для фигурного обрезания секции
- CSS-анимации появления заголовка и кнопки (slide-in + fade)
- Hover-эффект на кнопке с ripple-псевдоэлементом через `::after`

---

## 🛠 Стек технологий

- HTML5
- CSS3 / SCSS
- Методология: BEM
- Flexbox
- Git & GitHub

## 📌 Планы по улучшению

**Главное**

- Ознакомиться с макетом и вёрстать оставшиеся секции

**SCSS**

- `_mixins.scss` пустой - наполнить или удалить
- `// fix` комментарии - убрать перед публикацией
- Вынести `@font-face` из `_variables.scss` в отдельный `_typography.scss`
- удалить лишние файлы шрифтов `.ttf` или подключить все нужные (подключены только 3)
- Шрифты `.ttf` → конвертировать в `.woff2`

**HTML / семантика**

- Добавить `<meta name="description">`, `<link rel="icon">`
- `alt` у логотипа на русском (`"Логотип воздушный шар"`) при `lang="en"` - привести к одному языку
- `title="Home"` у логотипа и `alt` дублируют друг друга - убрать `title`

**Адаптивность**

- Реализовать для трёх брейкпоинтов
- `height: 50rem` у `.container` - фиксированная высота, убрать
